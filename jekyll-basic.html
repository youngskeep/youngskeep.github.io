jekyll 기본 설정

항목 설명 비고
대상자 블로그 개설이 하고 싶은 개발자 분들  
누가 도움이 될까? 1. github.io 블로그를 구성해보고 싶은 분들
2. 만들었는데, 뭔가 이상하고 잘 안되는 분들
 
전달 메세지 저도 관련 글이나, 참고해서 만들었다가, 초기 환경은 구성했는데,
단순한 layout만 구성되고 _post 작성만 가능하였는데,
좀 제대로 해보려고 진행했던 내용을 공유합니다.		  

참고했던 유튜브 영상 정보

  • 얼큰우동TV, 쉽게배우는 IT

    • 유트브
      https://www.youtube.com/watch?v=2ClW2LdqP30&list=PL7nkwz9MkASx1wxXK51n7KtwQyXgoNL70
    • 저도 해당 영상을 참조해서 구성했습니다.
    • 장점
      • 강사분이셔서 친절하게 잘 설명해주신다.
      • 차분히 초급 개발자가 들어도, 전반적으로 도움되는 부분이 많이 있다.
      • CI부분 / js에 대한 minify / 폰트 및 스타일등에 대한 부분에 언급해주신다.
    • 단점
      • 단점은 특별히 없다. 강추 드린다.!!

  • 테디노트

    • 유트브 https://www.youtube.com/watch?v=ACzFIAOsfpM
    • local환경에 ruby설치 없이 정말 10분만에 구성할 수 있게 설명해주십니다.
    • 이 영상도 도움이 되실거 같습니다.

  • BLEX

    • 유트브 https://www.youtube.com/watch?v=LyQgkZX2ZaM
    • 단순히 jekyll이 문법 / 동작원리등을 느낄수 있게 설명해주십니다.
    • jekyll은 감만 잡으시면, 다양하게 css / javasscript액션을 주실수 있습니다.
    • jekyll에서도 DB는 없지만 _config.yml 및 Develope이 필요한 사항임을 알게 됩니다.

  • 참고사항

    • tistory : 가장 보편화된 블로그
      많은 사용자들이 사용하며, 애드센스도 추가 가능하고 개발자의 Customize도 가능하다.
      기존적인 template도 있고, 괜찮은것 같지만 예전에 사용했던 기억도 있고 좀 더 다른 플랫폼에서 글작성을 새로 하고 싶은 생각도 있었다.
      찾아본 템플릿 중에 기본적으로 제공해주는것 말고 아래의 템플릿을 적용하면 깔끔하게 구성도 가능할거 같다.
      • 적용된 블로그 : https://cholol.tistory.com/506
      • 사용된 템플릿 : https://www.jbfactory.net/ (5만원이지만, tistory를 사용하려고 하는 사용자는 고려해볼만 할 것 같다.)
    • velog : 너무 깔끔한 Layout
      가장 눈에 들어온 블로그는 velog이다. 개발 관련된 많은 내용들이 가독성이 매우 높게 직관적으로 글을 작성 할수 있게 구성되어있다.
      내용을 찾아보니, 개발자 한분이 원래 기존에 서비스되는 블로그를 이용하시다가 직접 개발을 해서 서비스로 오픈하신 것을 알수 있었다.
      하지만, 여기에는 구글 ad-sense가 적용이 되지 않는다.
